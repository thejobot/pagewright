Set up studio

Studio needs a GitHub token to read and write your repo. It's stored only in your browser and sent with each request. Nothing touches a file on disk.

1
Create a fine-grained PAT
Open GitHub with the right scope pre-suggested:
Open GitHub → New PAT ↗
Repository access: Only select repositories → your repo
Permissions: Contents Read & write · Pull requests Read & write
Expiration: 30–90 days is fine.
2
Paste the token here
Starts with github_pat_ (fine-grained) or ghp_ (classic). Both work. Use if you need to rotate.
